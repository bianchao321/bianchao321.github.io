第一部分
1,安装hexo
npm install -g hexo-cli
2,初始项目(找一个空的文件夹)
hexo init blog
cd blog //进入这个myblog文件夹
npm install

    node_modules: 依赖包
    public：存放生成的页面
    scaffolds：生成文章的一些模板
    source：用来存放你的文章
    themes：主题        
3,启动blog
    hexo g
    hexo server

4,GitHub创建个人仓库
    创建一个仓库，名称为yourname.github.io
5,将hexo部署到GitHub
    1) 打开站点配置文件 _config.yml
    2) 在文件末尾修改
        deploy:
            type: git
            repo: https://github.com/YourgithubName/YourgithubName.github.io.git
            branch: master
    3) npm install hexo-deployer-git --save
    4) hexo clean      清除了你之前生成的东西
    hexo generate   生成静态文章
    hexo deploy     部署文章
    最后一个命令执行完后会弹出一个弹窗，输入github的账号密码即可
    5) 进入 http://yourname.github.io 检测blog是否可以访问

第二部分
hexo的基本配置，更换主题，实现多终端工作，以及在coding page部署实现国内外分流
1. hexo基本配置
_config.yml文件属性介绍：

    title：         网站标题
    subtitle:        网站副标题
    description:    网站描述
    author:            您的名字
    language:        网站使用的语言
    timezone:        网站时区。Hexo 默认使用您电脑的时区。时区列表。比如说：America/New_York, Japan, 和 UTC 







hexo clean
hexo g
hexo d
